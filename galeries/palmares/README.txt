CONVENTION DE NOMMAGE :
-----------------------
  Le numero en debut de nom definit l'ordre d'affichage.
  Les images sont associees aux cartes du palmares dans
  l'ordre (1ere image -> 1ere carte, etc.).

EXEMPLES :
----------
  01.jpg   -> Image de la 1ere distinction
  02.png   -> Image de la 2eme distinction
  03.jpg   -> Image de la 3eme distinction

NOTE :
------
  Pour ajouter une nouvelle distinction, il faut aussi
  ajouter la carte correspondante dans evenement.html
  (annee, médaille, titre, description) avec l'attribut
  data-galerie-img="palmares/{index}" sur le <img>.
