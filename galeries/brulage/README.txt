CONVENTION DE NOMMAGE :
-----------------------
  Chaque image represente une etape du processus.
  Le numero definit l'ordre, le mot apres le tiret
  definit le label affiche sous l'image.

EXEMPLES :
----------
  01-avant.jpg      -> 1ere etape, label "Avant"
  02-pendant.jpg    -> 2eme etape, label "Pendant"
  03-pendant.png    -> 3eme etape, label "Pendant"
  04-resultat.jpg   -> 4eme etape, label "Resultat"

LABELS DISPONIBLES :
--------------------
  Vous pouvez utiliser n'importe quel mot comme label :
  avant, pendant, apres, resultat, etc.
  Le label est le mot apres le tiret dans le nom du fichier.
