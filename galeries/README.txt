              Cathy Coiffure - Mode d'emploi

COMMENT CA MARCHE ?
-------------------
Chaque sous-dossier ici correspond a une galerie sur le site.
Il suffit de glisser des images dans le bon dossier.
Les images apparaitront automatiquement sur la bonne page !


DOSSIERS DISPONIBLES :
----------------------

  Coiffures :
    dames/          -> Page "Coiffures Dames" (avant/apres)
    messieurs/      -> Page "Coiffures Messieurs" (avant/apres)
    mariees/        -> Page "Mariees & Soirees", section Mariees
    soirees/        -> Page "Mariees & Soirees", section Soirees
    aines/          -> Page "Nos aines"

  Techniques :
    brulage/        -> Page "Techniques", section Brulage (etapes)
    colorations/    -> Page "Techniques 2", section Colorations (avant/apres)
    permanentes/    -> Page "Techniques 2", section Permanentes (etapes)
    ondulations/    -> Page "Techniques 2", section Ondulations (avant/apres)
    balmain/        -> Page "Extensions Balmain"

  Autres :
    vitrine/        -> Page "Vitrine"
    palmares/       -> Page "Palmares & Evenements"


3 TYPES DE GALERIES :
---------------------

  1. GALERIE SIMPLE (mariees, soirees, aines, vitrine, balmain)
     -> Nommer : 01.jpg, 02.jpg, 03.png...
     -> Pour une image grande : 01-large.jpg

  2. AVANT / APRES (dames, messieurs, colorations, ondulations)
     -> Nommer : 01-avant.jpg + 01-apres.jpg (par paires)

  3. ETAPES (brulage, permanentes)
     -> Nommer : 01-avant.jpg, 02-pendant.jpg, 03-resultat.jpg
     -> Le mot apres le tiret = label affiche


ETAPES POUR AJOUTER DES IMAGES :
---------------------------------
  1. Nommer l'image selon la convention du dossier
  2. Glisser l'image dans le bon dossier
  3. C'est fait !


FORMATS ACCEPTES :
------------------
  .jpg  .jpeg  .png  .webp  .gif  .avif
