CONVENTION DE NOMMAGE :
-----------------------
  Chaque image represente une etape du processus.
  Le numero definit l'ordre, le mot apres le tiret
  definit le label affiche sous l'image.

EXEMPLES :
----------
  01-avant.jpg      -> 1ere etape, label "Avant"
  02-pendant.jpg    -> 2eme etape, label "Pendant"
  03-apres.jpg      -> 3eme etape, label "Apres"
